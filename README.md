* :python:       3.10
* :pip:          1.5.6
* :virtualenv:   1.11.6
* :numpy:

## install and start

clone form github and install in virtualenv::

    ```bash
    #!/bin/bash
    git clone
    cd
    virtualenv .venv
    source .venv/bin/activate
    pip install .
    ```

    '''cmd
    git clone
    cd
    virtualenv .venv
    .venv\Scripts\activate
    pip install .
    '''
